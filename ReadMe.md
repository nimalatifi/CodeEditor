NIMACodeEditor v 1.0

This a Qt based Code editor 

## Requirements
0. C++11 featured compiler.
0. Qt 5.13.0.

## Abilities
1. Auto close bracketing.
2. syntax highlighting.(based on exprTK librari's keywords)
3. Auto indentation.
4. bracket matching
5. commenting
